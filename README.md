# Quick Chat - Backend API 

A simple and efficient backend API built with Node.js, Express, and MongoDB (Mongoose) for handling message processing and chat operations.
This project was created to demonstrate backend development skills, API structuring, database operations, and server handling.

# Features

- RESTful API with Express  
- MongoDB database integration using Mongoose  
- Chat/message handling endpoints  
- Error handling & async operations  
- Clean project structure  
- Easy to expand with frontend or UI later  

# Tech Stack

Backend: Node.js, Express  
Database: MongoDB (Mongoose ODM)  
Version Control: Git & GitHub
